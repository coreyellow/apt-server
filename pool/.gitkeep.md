# APT Repository Pool

This directory contains all the Debian package files (.deb) that are part of this APT repository.

## Structure

Packages are organized by distribution and component:
- `stable/main/`: Stable distribution packages
- `test/main/`: Test distribution packages

To add a package to the repository:
1. Upload your .deb file to the appropriate directory:
   - `pool/stable/main/` for stable packages
   - `pool/test/main/` for test packages
2. Commit and push the changes
3. The GitHub Actions workflow will automatically update the repository metadata
